# unsupervised-customer-segmentation

## Overview
This project focuses on customer personality segmentation using Unsupervised Machine Learning techniques. The goal is to group customers based on their purchasing behavior, income, spending patterns, and demographics to help businesses improve marketing strategies and customer targeting.

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Customer segmentation using K-Means Clustering
- PCA for dimensionality reduction
- Data visualization
- Cluster analysis and insights

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Dataset
The dataset contains customer demographic and spending information such as:
- Age
- Income
- Education
- Marital Status
- Spending habits
- Purchase history

---

## Machine Learning Techniques
### 1. K-Means Clustering
Used to group customers into different segments.

### 2. PCA (Principal Component Analysis)
Used for dimensionality reduction and visualization.

---

## Project Workflow
1. Import dataset
2. Data preprocessing
3. Exploratory Data Analysis
4. Feature scaling
5. Apply K-Means clustering
6. Visualize clusters
7. Analyze customer segments

---

## Results
The model successfully segmented customers into different groups based on behavioral patterns. These insights can help businesses:
- Improve targeted marketing
- Understand customer behavior
- Increase customer retention
- Optimize sales strategies
